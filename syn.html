Starting To make the full syntax (July 2025)

This is when I REALLY started to think about the syntax of the language
Wrote it down tried it out, checking if it feels right to me
I was deciding between indentation and curly braces, though braces were easy to impliment, indentation looked cleaner so that won
There was a debate in mind whether I should add the ':' like python or not
Although it looked more defining, I decided not to and went for almost no visual clutter
This was the time where I decided my langauge would be statically written to make the performance better
This meant, instead of 'a = 5' or 'var a = 5', I went for 'int a = 5'
I also defined literal types : Ingegers (int), Decimals (dec), Texts (text), lists (list), dictionaries (dict), and decided on more later
Although I made the langauges directed towards game developers (more towards indie developers), The langauge was not ONLY meant for them and would be used by others
This langauge's motto (to me) became 'Making it simple'
